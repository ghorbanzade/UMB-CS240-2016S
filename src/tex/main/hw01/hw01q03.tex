\section*{Question 3}

Write a program \texttt{fahrenheit.c} that prints Fahrenheit equivalents of Celcius temperatures 60, 70, 80, 90 and 100 in the following format.

\begin{terminal}
 C    F
---------
 60   140
 70   158
 80   176
 90   194
 100  212
\end{terminal}
