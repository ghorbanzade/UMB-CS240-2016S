\section*{Question 1}

Write a program \texttt{hello-world.c} that prints \textit{``Hello World!"} on the console once executed, as shown below.

\begin{terminal}
$ gcc hello-world.c -o hello-world
$ ./hello-world
Hello World!
\end{terminal}
